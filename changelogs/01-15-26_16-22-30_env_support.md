# Changelog: .env File Support

**Date:** January 15, 2026

**Description:** Added support for .env file to load environment variables

## Changes Made

- Created `.env.example` template file with all environment variables
- Simplified `src/config.py` to use pydantic-settings Field() defaults
- Updated `src/main.py` to use new field names (proxy_host, proxy_port)
- Updated `description.md` with environment variable documentation

## Bug Fixes

- Fixed ALLOWED_ORIGINS format in .env and .env.example from `*` to `["*"]` (pydantic-settings requires JSON format for List[str] fields)

## Files Changed

- `.env`
- `.env.example`
- `src/config.py`
- `src/main.py`
- `description.md`

## Testing Notes

Server running successfully with new configuration
